# 外研2025版三上字母拼写练习软件
专为外研版2025版三年级上册英语教材设计的无水印字母拼写练习工具，聚焦26个英文字母基础学习。

## 📌 核心特点
- ✅ 无水印、无广告：纯学习功能，无任何干扰元素；
- ✅ 教材适配：精准匹配外研2025版三上英语字母学习要求；
- ✅ 功能完整：字母发音、随机出题、答案校验、分数统计一站式练习；
- ✅ 操作简单：界面简洁，适配小学生使用习惯。

## 🎯 核心功能
### 1. 随机字母出题
软件随机展示26个字母的大写/小写形式，避免固定顺序练习的记忆依赖，强化字母识别能力。

### 2. 字母发音播放
配套字母标准发音（需自行配置音频文件），点击「播放发音」即可聆听，兼顾听、写、认能力训练。

### 3. 答案即时校验
输入字母后点击「检查答案」，软件自动判定对错并给出反馈，即时纠正拼写错误。

### 4. 练习分数统计
自动记录练习得分与总题数，支持分数重置，方便跟踪学习进度。

## 📋 使用指南
### 1. 环境准备
- 安装Python 3.8及以上版本；
- 安装依赖：`pip install pygame -i https://pypi.tuna.tsinghua.edu.cn/simple`。

### 2. 音频配置（可选）
- 在软件目录下创建`audio`文件夹；
- 将字母发音文件（mp3/wav格式）放入该文件夹，命名为`a.mp3`、`A.mp3`、`b.mp3`等。

### 3. 运行软件
```bash
python alphabet_spelling.py
